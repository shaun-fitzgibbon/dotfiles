# My Dotfiles setup

## things to install

### Run chezmoi script to install everything

```sh
sh -c "$(curl -fsLS get.chezmoi.io)" -- init --apply $GITHUB_USERNAME

or

brew install chezmoi
chezmoi init --apply https://github.com/$GITHUB_USERNAME/dotfiles.git
```
